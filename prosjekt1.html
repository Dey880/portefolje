CHATTER'N

Chatter'n er en chatapp laget som et skoleprosjekt for å konkurrere med populære plattformer som Discord. Den benytter Node.js for backend, React for frontend, og MongoDB for lagring av data som brukere, chatrom og meldinger. Jeg har også implementert en API for å generere profilbilder automatisk.

Teknologier brukt:

  • CSS, JavaScript
  • Node.js
  • React
  • MongoDB
  • SocketIo
  • GitHub

Utfordringer og løsninger:

En av de største utfordringene var å håndtere profilbildene som ble hentet fra et API. Jeg måtte sikre at bildene ble lastet ned korrekt og lagret på serveren, samt at de ble slettet hvis brukeren valgte å bytte bilde. Dette krevde at jeg implementerte en funksjon som både hentet bildene fra API-et og fjernet de gamle bildene når et nytt bilde ble lastet opp, for å unngå unødvendig lagring og sikre at brukerens profil alltid var oppdatert.
En annen utfordring var å implementere live chat-funksjonen. Her brukte jeg SocketIo for å tillate sanntidskommunikasjon mellom brukere.
