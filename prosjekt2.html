Guide-nettside

Guide-nettsiden er utviklet for å gi brukere muligheten til å lage, dele og lese guider om ulike emner. Siden benytter en enkel og intuitiv design, som gjør det lett for brukere å lage innhold ved hjelp av et tekstredigeringsverktøy og deretter publisere det for andre å se. Jeg har også implementert funksjoner for kategorisering og søk, slik at brukere enkelt kan finne guider som passer deres interesser.

Teknologier brukt:

  • HTML, CSS, JavaScript
  • Node.js (for backend)
  • MongoDB (for lagring av guider og brukerdata)
  • Express.js (for server-side logikk)
  • GitHub

Utfordringer og løsninger:

En av de største utfordringene var å implementere en effektiv lagring og søkefunksjon for guider. For å løse dette, brukte jeg MongoDB til å lagre både guider og metadata (som kategorier og tags), og implementerte et søkeverktøy som benyttet tekstsøk i MongoDB for rask tilgang til spesifikke guider. En annen utfordring var å sørge for at innholdet som ble publisert, ble gjort trygt og sikkert, noe som krevde implementering av grunnleggende brukerautentisering og sanitizing av input for å forhindre sikkerhetsproblemer.
